# Pipeline Process

1. Build
    * Install Node.js 14.15
    * Install Front-End Dependencies
    * Install API Dependencies
    * Front-End Lint
    * Front-End Build
    * API Build
2. Approve
    * Hold pipeline until approved
3. Deploy
    * Setup Elastic Beanstalk
    * Install AWS CLI
    * Deploy API and Front-End
